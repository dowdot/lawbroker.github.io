加入Email notifications 的 SMTP Server　設定

EasyRedmine實際安裝時，建立cate table時一直有遇到 Mysql2::Error: Index column size too large. The maximum column size is 767 bytes.錯誤訊息。網路上有提到，將varchar 的長度限制為191就可以解決。但後來Create Table時用的是DEFAULT CHARACTER SET utf8 DEFAULT COLLATE utf8_general_ci，
安裝就過關了，所以不確定是誰起了作用。所以，備註一下，如果卡關，就加一下設定。
